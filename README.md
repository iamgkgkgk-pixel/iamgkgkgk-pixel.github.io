# 田园时光 - 3D农场养成游戏

一款基于 Three.js 的 3D 农场养成经营游戏。

## 🎮 游戏特色

- 🌱 11种作物种植系统（速熟/普通/珍稀/传说）
- 🐄 8种动物养殖与亲密度系统
- 🏆 17个成就与任务系统
- 📅 7天签到循环奖励
- 💰 离线收益计算（最多8小时）
- 🌤️ 昼夜变化与天气系统
- 💾 自动存档（30秒保存一次）

## 🚀 在线体验

部署后访问 Vercel 提供的域名即可直接游玩，无需安装任何插件。

## 🛠️ 本地运行

直接用浏览器打开 `index.html` 即可（推荐 Chrome/Edge）。

## 📁 项目结构

```
farm3d/
├── index.html          # 主页面
├── vercel.json         # Vercel 部署配置
└── js/
    ├── main.js         # 游戏入口
    ├── gameData.js     # 游戏数据配置
    ├── gameState.js    # 状态管理
    ├── scene3d.js      # Three.js 3D场景
    ├── farmLogic.js    # 农场核心逻辑
    ├── ui.js           # 界面渲染
    └── adSystem.js     # 广告系统
```

## 📜 隐私政策

本游戏不收集任何个人信息，游戏数据仅保存在本地浏览器中。
