### 转盘"再转一次"Bug修复

**修复的3个问题：**
1. **转盘不转动**：原来每次 targetAngle 固定值（360*5+offset），第二次旋转 CSS transition 不触发。改为累积角度 `_wheelCurrentAngle`，每次在前一次基础上叠加旋转
2. **transition 未重触发**：添加 `disc.style.transition='none'` + `offsetHeight` force reflow + 重新设置 transition 的三步法
3. **防连点**：添加 `_wheelSpinning` 标志位防止旋转中重复点击
4. **状态文字不更新**：给底部状态div添加 `id="wheel-status-text"`，旋转结束后调用 `_updateWheelStatus()` 动态更新
