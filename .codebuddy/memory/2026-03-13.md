### 音乐音效系统（音效大师全面分析与实现）

**系统架构：** 100% Web Audio API 程序化合成，零外部音频文件依赖
**新建文件：** `js/audioSystem.js`（约1413行）

**音效分析与覆盖范围：**

1. **BGM背景音乐**
   - 田园风格五声音阶旋律（C大调五声：C-D-E-G-A）
   - 6条旋律片段轮播 + 8组和弦进行
   - 三层音色叠加：正弦波（主旋律）+ 三角波（温暖底色）+ 节奏层（低音鼓+木块）
   - 每小节8拍×0.45s自动调度下一小节

2. **农场操作音效（6种）**
   - 🌱种植：泥土噪声 + 种子落地叮 + 生长暗示上升音
   - 💧浇水：带通滤波噪声模拟水流 + 随机频率水滴声
   - 🌿施肥：沙沙撒落噪声 + 正面反馈音
   - 🌾收获：拔起声 + 五声音阶上升琶音 + 金币叮铃
   - ✨金色收获：五度和弦齐奏 + 泛音层 + 闪烁高频
   - ⚡加速卡：低频蓄力 + 锯齿波扫频上升 + 确认方波
   - 🏗️升级土地：锤击噪声×2 + 升级完成三音阶

3. **动物互动音效（5种）**
   - 🥕喂食：倒食物噪声 + 咀嚼低频脉冲×4 + 开心音
   - 🤝抚摸：温暖上行三度（G4→B4→D5）+ 心跳声
   - 🥚收集产出：拿起声 + 金币叮当 + 满足音
   - ✂️剪羊毛：高频噪声脉冲×3（剪刀声）+ 确认音
   - 🐄动物叫声：按类型切换（牛=低沉锯齿波"哞"，鸡=尖锐方波"咯咯"，羊=三角波"咩"，鸭=锯齿波"嘎嘎"）

4. **UI交互音效（8种）**
   - 按钮点击、菜单打开/关闭、通知弹出、警告、错误、金币获得、购买收银机、签到琶音、工具切换

5. **特殊系统音效（9种）**
   - 扭蛋：投币金属声 + 机械转动 / 揭晓（普通/稀有琶音/传说史诗和弦+闪烁高频）
   - 钓鱼：抛竿扫频+入水噪声 / 鱼咬钩急促叮叮 / QTE命中 / 钓到鱼胜利音 / 失败下行
   - 升级：华丽8音上行琶音+结束和弦
   - 成就：号角式方波上行（G4→C5→E5→G5）
   - 一键出售：连续金币声+收银机叮

6. **环境音系统**
   - 白天：随机鸟鸣（2000-4000Hz正弦脉冲，2秒间隔30%概率触发）
   - 夜晚：蟋蟀声（4000-5000Hz连续脉冲×6，1.5秒间隔40%概率）
   - 雨天：低通滤波噪声循环（小雨2000Hz/大雨3000Hz）
   - 雷暴：雨声+低频锯齿波隆隆声（8秒间隔15%概率）
   - 雪天/暴风雪：带通400Hz噪声模拟风声

7. **HUD控制面板**
   - 右下角🔊按钮展开4通道独立音量滑块（总音量/音乐/音效/环境）
   - 一键静音 + BGM播放/暂停开关
   - localStorage持久化音量偏好

**集成方式：** Monkey-patch挂载（不修改原函数代码，只在外层包装追加音效调用）
