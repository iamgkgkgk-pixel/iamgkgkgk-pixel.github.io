# 田园时光 - 3D农场养成游戏 项目记忆

> 最后更新：2026-03-13

## 项目概况
- **名称**：田园时光（Farm3D）
- **类型**：基于 Three.js 的 3D 农场养成经营游戏
- **技术栈**：纯前端，HTML + 原生 JS + Three.js（CDN），无打包工具
- **部署**：Vercel + GitHub Pages，仓库 `iamgkgkgk-pixel/iamgkgkgk-pixel.github.io`
- **数据存储**：localStorage（`farm3d_save`），无后端
- **广告**：Google AdSense + 模拟广告SDK

## 项目结构
```
├── index.html          # 主页面（含全部CSS和HTML结构，约56KB）
├── vercel.json         # Vercel 部署配置
├── README.md
└── js/
    ├── main.js             # 游戏入口（加载/主循环/快捷键/页面生命周期）
    ├── gameData.js         # 数据配置（作物/动物/工具/成就/任务/天气/季节）
    ├── gameState.js        # 状态管理（玩家数据/存档/离线进度/升级/任务）
    ├── scene3d.js          # 3D场景核心（渲染器/相机/地形/动物建模/点击检测）
    ├── sceneBeautify.js    # 场景美化（草地/路径/围栏/装饰物/氛围粒子）
    ├── sceneHorizon.js     # 远景系统（雾效/远山/飞鸟/天地融合）
    ├── farmLogic.js        # 农场业务逻辑（种植/浇水/收获/动物操作/商店/签到）
    ├── ui.js               # UI管理（HUD/商店/背包/任务/成就/通知）
    ├── plants.js           # 植物3D模型（11种作物×5阶段生长/摇摆动画）
    ├── pond.js             # 池塘系统（水面/岸边装饰/涟漪/鱼影/蜻蜓/青蛙）
    ├── fishing.js          # 钓鱼小游戏（状态机/QTE/15种鱼/碎片合成）
    ├── weatherSystem.js    # 天气视觉（9种天气/雨雪粒子/昼夜/彩虹/极光/雷暴）
    ├── animalBehavior.js   # 动物行为AI（5种专属行为/天气响应/情感系统/Juice反馈）
    ├── gachaSystem.js      # 扭蛋机系统（3D扭蛋机/4类物品/珍藏品/代币获取）
    ├── mysterySeeds.js     # 神秘种子盲盒（3种袋/变异/悬念揭晓/新手保底）
    ├── pokedex.js          # 图鉴系统（5大类/96条目/里程碑奖励）
    └── adSystem.js         # 广告系统（横幅/插屏/激励视频/模拟SDK）
```

## 已实现的核心功能
1. **种植系统**：11种作物（速熟/普通/珍稀/传说），5阶段生长，浇水/施肥加速
2. **动物系统**：8种动物（鸡/鸭/牛/羊/猪/羊驼/孔雀/独角兽），差异化行为AI
3. **3D场景**：程序化建模，美化地形/围栏/路径，昼夜变化，远景系统
4. **天气系统**：9种天气视觉效果，雨雪粒子，彩虹/极光/雷暴/流星雨
5. **池塘与钓鱼**：不规则池塘+钓鱼小游戏（15种鱼，QTE玩法），碎片合成
6. **扭蛋机**：3D可交互扭蛋机，4类物品，五步动画，珍藏品系统
7. **神秘种子盲盒**：3种种子袋，变异系统（颜色/巨型/迷你/闪光），悬念揭晓
8. **图鉴系统**：5大类96个条目（植物/动物/鱼类/成就/珍藏品），里程碑奖励
9. **任务与成就**：每日/每周/主线任务，17个成就
10. **签到系统**：7天循环，连续签到奖励
11. **离线收益**：最多8小时，作物生长+动物产出
12. **广告变现**：横幅/插屏/激励视频，模拟SDK+真实AdSense预留

## 模块依赖关系
```
用户点击 → Scene3D(射线检测) → farmLogic(业务逻辑)
                                    ├→ GameState(数据持久化)
                                    ├→ Scene3D(3D视觉更新)
                                    ├→ UI(2D界面更新)
                                    ├→ MysterySeeds(盲盒)
                                    ├→ AnimalBehavior(动物AI)
                                    ├→ FishingSystem(钓鱼)
                                    ├→ GachaSystem(扭蛋)
                                    └→ Pokedex(图鉴)
```
